
### Prerequisites

- Python 3.8 or higher
- A Groq API key (free at [console.groq.com](https://console.groq.com))

### Installation

1. **Clone the repository**
```bash
   git clone https://github.com/narapuGgava/russian-tutor-torfl-a1.git
   cd russian-tutor-torfl-a1
```

2. **Install dependencies**
```bash
   pip install -r requirements.txt
```

3. **Set up your API key**
   
   Option A - Environment variable (recommended):
```bash
   export GROQ_API_KEY="your-api-key-here"
```
   
   Option B - The app will prompt you to enter it on first run

4. **Run the application**
```bash
   streamlit run russian_final_bot.py
```

5. **Open in browser**
   
   The app will automatically open at `http://localhost:8501`

## 📖 How to Use

### First Time Setup
1. Start the app and enter your Groq API key (if not set as environment variable)
2. Begin chatting with your tutor to learn your first words
3. The tutor will automatically save new vocabulary when teaching
